# apcupsd

The official apcupsd project is at https://sourceforge.net/projects/apcupsd/.

This "fork" will provide an unofficial modern build of apcupsd.

This project is a work-in-progress.

Currently, "make" builds only gapcmon (apcupsd-gui).

gapcmon has been migrated from GConf to GSettings.

gapcmon is being migrated to GTK 3.

gapcmon will be migrated to GTK 4.
